# HNG Stage 1 API

A simple REST API built with ASP.NET Core .NET 8.

## How to run locally
```bash
dotnet run
```

## Endpoints
- `GET /` → `{"message": "API is running"}`
- `GET /health` → `{"message": "healthy"}`
- `GET /me` → returns name, email, github
